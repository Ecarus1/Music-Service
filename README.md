# CodeMasters study!

## Личный проект - "Новости"

## Что за проект?

Это приложение для просмотра актуальных новостей. В приложении удобно отсортировывать информацию по дате загрузки или по самым популярным.

## Настройка проекта

```
npm install
```

## Запуск проекта

```
npm start
```

## Веб дизайн

-   Базовый CSS

-   Bootstrap
